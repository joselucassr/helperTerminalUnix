@title ~$ cd ..

# Basicamente
Muda para o diretório pai.

# Variações

@group(cd)

# Mais a fundo
@mark("cd") significa "change directory", que pode ser traduzido para "mudar diretório", e em conjunto ao comando @ref(ls)"ls" é a principal forma de se mover entre diretórios.

# Exemplos:
@CodeArea
// Sair da pasta "/trybe/estudante" para a pasta "/trybe"
@mark("~/trybe/estudante$ cd") ..
@EndCodeArea
